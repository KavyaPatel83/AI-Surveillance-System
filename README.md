# AI Surveillance System 🚨

## Overview
This project detects suspicious human activities from video using deep learning and sends email alerts.

## Features
- Video-based activity detection
- Multi-class classification (12 classes)
- Email alert system
- Frame-based prediction using CNN + sequence model

## Files
- train.ipynb → model training
- detection.ipynb → detection + email alert
- classes.pkl → class labels

## Model
Model is stored in Google Drive due to size limitations.

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## How to Run
1. Open detection.ipynb in Google Colab
2. Upload video file
3. Run all cells
